jqmLiLink (directive in module jqm )

Description

Creates a jquery mobile list item link entry.

Must be inside of a jqmListview

  • Add a <img jqm-li-thumb> inside for a thumbnail.
  • Add a <span jqm-li-count> inside for a count.

Usage

as attribute
<ANY jqm-li-link
     [jqmLiLInk="{string}"]
     [icon="{string}"]
     [iconpos="{string}"]
     [iconShadow="{string}"]>
   ...
</ANY>

Parameters

  • jqmLiLInk(optional) – {string=} –

    The link, or href, that this listitem should go to when clicked.

  • icon(optional) – {string=} –

    What icon to use for the link. Default 'ui-icon-arrow-r'.

  • iconpos(optional) – {string=} –

    Where to put the icon. Default 'right'.

  • iconShadow(optional) – {string=} –

    Whether the icon should have a shadow or not. Default true.
