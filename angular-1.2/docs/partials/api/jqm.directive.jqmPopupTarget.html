jqmPopupTarget (directive in module jqm )

Description

Marks an element as a target for a jqmPopup, and assigns a model to toggle to show or hide that popup on the element.

See jqmPopup for an example.

Usage

as attribute
<ANY jqm-popup-target="{expression}"
     [jqm-popup-model="{expression}"]
     [jqm-popup-placement="{string}"]>
   ...
</ANY>

Parameters

  • jqmPopupTarget – {expression} –

    Model of a jqmPopup that this element will be linked to.

  • jqm-popup-model(optional) – {expression=} –

    Assignable angular boolean expression that will say whether the popup from jqmPopupTarget is opened on this element. Default '$popup'.

  • jqm-popup-placement(optional) – {string=} –

    The placement for the popup to pop over this element. Overrides jqmPopup's placement attribute. See jqmPopup for the available values.
