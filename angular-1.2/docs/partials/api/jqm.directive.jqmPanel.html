jqmPanel (directive in module jqm )

Description

Creates a jquery mobile panel. Must be placed inside of a jqm-panel-container.

Usage

as attribute
<ANY jqm-panel
     [opened="{expression}"]
     [display="{string}"]
     [position="{string}"]>
   ...
</ANY>

Parameters

  • opened(optional) – {expression=} –

    Assignable angular expression to data-bind the panel's open state to.

  • display(optional) – {string=} –

    Default 'reveal'. What display type the panel has. Available: 'reveal', 'overlay', 'push'.

  • position(optional) – {string=} –

    Default 'left'. What position the panel is in. Available: 'left', 'right'.
