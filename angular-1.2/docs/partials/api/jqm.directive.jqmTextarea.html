jqmTextarea (directive in module jqm )

Description

Creates an jquery mobile textarea on the given elemen.

Usage

as attribute
<ANY jqm-textarea
     ngModel="{string}"
     [disabled]>
   ...
</ANY>

Parameters

  • ngModel – {string} –

    Assignable angular expression to data-bind to.

  • disabled(optional) – {string=} –

    Whether this input is disabled.
