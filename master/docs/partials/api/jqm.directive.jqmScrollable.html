jqmScrollable (directive in module jqm )

Description

Overview

jqmScrollable enables fake scrolling for the given element using angular-scrolly.

Usage

as element:
<jqm-scrollable>
</jqm-scrollable>
as attribute
<ANY jqm-scrollable>
   ...
</ANY>
as class
<ANY class="jqm-scrollable">
   ...
</ANY>
