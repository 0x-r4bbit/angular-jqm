jqmPopup (directive in module jqm )

Description

Creates a popup with the given content. The popup can be opened and closed on an element using jqmPopupTarget.

Tip: put a jqmView inside a popup to have full scrollable pages inside. 

<div jqm-popup="myPopup">
  <div jqm-view="{ 
    templateUrl: 'views/my-popup-content-page.html', 
    controller: 'MyPopupController'
  }"></div>
</div>

Usage

as attribute
<ANY jqm-popup="{expression}"
     [animation="{expression}"]
     [placement="{expression}"]
     [overlay-theme="{expression}"]
     [corners="{expression}"]
     [shadow="{expression}"]>
   ...
</ANY>

Parameters

  • jqmPopup – {expression} –

    Assignable angular expression to bind this popup to. jqmPopupTargets will point to this model.

  • animation(optional) – {expression=} –

    jQuery Mobile animation to use to show/hide this popup. Default 'fade'.

  • placement(optional) – {expression=} –

    Where to put the popup relative to its target. Available: 'left', 'right', 'top', 'bottom', 'inside'. Default: 'inside'.

  • overlay-theme(optional) – {expression=} –

    The theme to use for the overlay behind the popup. Defaults to the popup's theme.

  • corners(optional) – {expression=} –

    Whether the popup has corners. Default true.

  • shadow(optional) – {expression=} –

    Whether the popup has shadows. Default true.
