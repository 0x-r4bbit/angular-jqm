jqmFlip (directive in module jqm )

Description

Creates a jquery mobile flip switch on the given element.

Anything inside the jqm-flip tag will be a label.

Labels for the on and off state can be omitted. If no values for the on and off state are specified on will be bound to true and off to false.

A theme can be set with the jqm-theme directive and specific styles can be set with the ng-style parameter. This is necessary to extend the width of the flip for long labels.

Usage

as attribute
<ANY jqm-flip
     [ngModel="{expression}"]
     [disabled]
     [mini="{string}"]
     [ngOnLabel="{string}"]
     [ngOnValue="{string}"]
     [ngOffLabel="{string}"]
     [ngOffValue="{string}"]>
   ...
</ANY>

Parameters

  • ngModel(optional) – {expression=} –

    Assignable angular expression to data-bind to.

  • disabled(optional) – {string=} –

    Whether this flip switch is disabled.

  • mini(optional) – {string=} –

    Whether this flip should be displayed minified.

  • ngOnLabel(optional) – {string=} –

    The label which should be shown when fliped on (optional).

  • ngOnValue(optional) – {string=} –

    The value to which the expression should be set when fliped on (optional, default: true).

  • ngOffLabel(optional) – {string=} –

    The label which should be shown when fliped off (optional).

  • ngOffValue(optional) – {string=} –

    The value to which the expression should be set when fliped off (optional, default:false).
