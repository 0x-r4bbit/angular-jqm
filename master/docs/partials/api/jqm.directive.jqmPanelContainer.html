jqmPanelContainer (directive in module jqm )

Description

A container for jquery mobile panels.

If you wish to use this with a view, you want the jqm-panel-container as the parent of your view and your panels. For example: 

<div jqm-panel-container="myPanel">
  <div jqm-panel>My Panel!</div>
  <div jqm-view></div>
</div>

Usage

as attribute
<ANY jqm-panel-container="{expression}">
   ...
</ANY>

Parameters

  • jqmPanelContainer(optional) – {expression=} –

    Assignable angular expression to data-bind the panel's open state to. This is either left (show left panel), right (show right panel) or null.
