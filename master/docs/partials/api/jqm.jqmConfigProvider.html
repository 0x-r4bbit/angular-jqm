jqmConfigProvider (service in module jqm )

Description

Used to configure the default theme.

Methods

  • primaryTheme(newTheme)

    Sets/gets the default primary theme (used if jqm-theme is not set on the element). Default: 'c'

    Parameters

    • newTheme(optional) – {string=} –

      The new primary theme to set.

    Returns

    {string} –

    The current primary theme.

  • secondaryTheme(newTheme)

    Sets/gets the secondary theme (used on footers, headers, etc if not theme is set on the element). Default: 'a'

    Parameters

    • newTheme(optional) – {string=} –

      The new secondary theme to set.

    Returns

    {string} –

    The current secondary theme.
