$loadDialog (service in module jqm )

Description

Shows a wait dialog to indicate some long running work.

Dependencies

  • $rootElement
  • $rootScope

Usage

$loadDialog();

Methods

  • hide()

    Restores the dialog state before the show function was called.

  • show(message, callback)

    Opens the wait dialog and shows the given message (if existing). If the user clicks on the wait dialog the given callback is called. This can be called even if the dialog is currently showing. It will then change the message and revert back to the last message when the hide function is called.

    Parameters

    • message(optional) – {string=} –

      The message to be shown when the wait dialog is displayed.

    • callback(optional) – {function=} –

      The Callback that is executed when the wait dialog is clicked.

  • waitFor(promise, message)

    Shows the dialog as long as the given promise runs. Shows the given message if defined.

    Parameters

    • promise – {Promise} –

      The Promise.

    • message(optional) – {string=} –

      The message to be show.

  • waitForWithCancel(The, cancelData, message)

    Same as jqm.$loadDialog#waitFor, but rejects the promise with the given cancelData when the user clicks on the wait dialog.

    Parameters

    • The – {Deferred} –

      deferred object to cancel the promise.

    • cancelData – {*} –

      To reject the promise with.

    • message(optional) – {string=} –

      The message to be show.
