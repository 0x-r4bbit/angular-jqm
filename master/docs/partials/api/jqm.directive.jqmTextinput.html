jqmTextinput (directive in module jqm )

Description

Creates an jquery mobile input on the given element.

Usage

as attribute
<ANY jqm-textinput
     ngModel="{string}"
     [type="{string}"]
     [disabled]
     [mini="{string}"]
     [clearBtn="{boolean}"]
     [clearBtnText="{string}"]
     [placeholder="{string}"]>
   ...
</ANY>

Parameters

  • ngModel – {string} –

    Assignable angular expression to data-bind to.

  • type(optional) – {string=} –

    Defines the type attribute for the resulting input. Default is 'text'.

  • disabled(optional) – {string=} –

    Whether this input is disabled.

  • mini(optional) – {string=} –

    Whether this input is mini.

  • clearBtn(optional) – {boolean=} –

    Whether this input should show a clear button to clear the input.

  • clearBtnText(optional) – {string=} –

    Defines the tooltip text for the clear Button. Default is 'clear text'.

  • placeholder(optional) – {string=} –

    Defines the placholder value for the input Element.
